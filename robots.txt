# robots.txt — agentpatterns.ai

# Three-tier AI crawler strategy:
#
#   TIER 1 — ALLOW (user-facing retrieval bots)
#     These bots power real-time citations and traffic. Allow unconditionally.
#
#   TIER 2 — DISALLOW (training scrapers)
#     These bots consume content for model training with no citation benefit.
#     Disallowed. This is a deliberate decision — revisit if policy changes.
#
#   TIER 3 — CDN-LEVEL BLOCK (non-compliant bots)
#     Bytespider (ByteDance) is documented to ignore robots.txt.
#     Block via Cloudflare WAF Custom Rule:
#       Field: User Agent | Operator: contains | Value: Bytespider
#       Action: Block
#     This cannot be expressed in robots.txt and must be set in the
#     Cloudflare Dashboard under Security > WAF > Custom Rules.
