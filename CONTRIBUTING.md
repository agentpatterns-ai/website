# Contributing

Thanks for your interest in improving Agent Patterns.

## How to contribute

Content in this repository is published automatically from a private source repo. We don't accept pull requests here, but we welcome your input through issues.

### Report an error

If you spot a factual error, broken link, or outdated information:

1. Open an [issue](https://github.com/agentpatterns-ai/website/issues/new)
2. Link to the page (URL or file path in this repo)
3. Describe what's wrong and, if possible, what the correct information is

### Suggest an improvement

For content improvements, restructuring, or rewording:

1. Open an [issue](https://github.com/agentpatterns-ai/website/issues/new)
2. Explain what you'd change and why

### Request a new page

Have an idea for a pattern, technique, or workflow that should be covered?

1. Open an [issue](https://github.com/agentpatterns-ai/website/issues/new)
2. Describe the topic and why it matters
3. Include links to sources if you have them

## What happens next

We triage issues and incorporate changes in the private source repo. When a new release is published, the updated content appears here and on [agentpatterns.ai](https://agentpatterns.ai).
